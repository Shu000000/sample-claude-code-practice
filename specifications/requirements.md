# 要件定義書

## プロジェクト概要

従業員のシフト作成を効率化するWebアプリケーション。紙・Excelによる手作業管理からの脱却を目的とし、管理者と従業員がそれぞれの権限に応じてシフトを作成・閲覧できる仕組みを提供する。

---

## 利用者

| ロール | 説明 |
|--------|------|
| 管理者 | 店長・責任者。シフト全体の作成・編集・確定を行う |
| 従業員 | アルバイト・パート含む一般社員。自分のシフト閲覧と希望提出を行う |

---

## 機能要件

### 認証・権限

| ID | 要件 |
|----|------|
| AUTH-01 | ログイン認証機能（ID/パスワード） |
| AUTH-02 | 管理者はシフト全体の閲覧・編集・削除が可能 |
| AUTH-03 | 従業員は自分のシフト閲覧と希望シフト提出のみ可能 |

### 管理者機能

| ID | 要件 |
|----|------|
| ADM-01 | 従業員の登録・編集・削除 |
| ADM-02 | シフトの作成・編集・削除 |
| ADM-03 | 日付・曜日ごとに勤務者を割り当て |
| ADM-04 | 勤務時間（開始・終了時刻）の設定 |
| ADM-05 | シフト一覧表示（月別・週別） |
| ADM-06 | シフトの確定・未確定の状態管理 |
| ADM-07 | CSV/Excel（xlsx）形式でのエクスポート |
| ADM-08 | 従業員の希望シフトを一覧で確認 |

### 従業員機能

| ID | 要件 |
|----|------|
| EMP-01 | 自分のシフトの閲覧 |
| EMP-02 | 希望シフトの提出 |
| EMP-03 | シフト確定後の確認 |
| EMP-04 | スマートフォンからの閲覧対応 |

---

## 画面一覧

| 画面ID | 画面名 | 対象ロール |
|--------|--------|-----------|
| SCR-01 | ログイン画面 | 全員 |
| SCR-02 | 管理者ダッシュボード | 管理者 |
| SCR-03 | シフト作成画面（カレンダー形式） | 管理者 |
| SCR-04 | 従業員一覧画面 | 管理者 |
| SCR-05 | 希望シフト入力画面 | 従業員 |
| SCR-06 | シフト確認画面 | 従業員 |
| SCR-07 | 希望シフト確認画面（管理者用） | 管理者 |

---

## API一覧

| API ID | API名 | 説明 |
|--------|-------|------|
| API-01 | 認証API | ログイン・ログアウト |
| API-02 | 従業員API | 従業員のCRUD |
| API-03 | シフトAPI | シフトのCRUD・確定状態管理 |
| API-04 | 希望シフトAPI | 従業員の希望シフト提出・取得 |
| API-05 | エクスポートAPI | CSV/Excelダウンロード |

---

## 非機能要件

| ID | 要件 |
|----|------|
| NFR-01 | PC・スマートフォン両対応（レスポンシブデザイン） |
| NFR-02 | クラウド上で動作するWebアプリケーション |
| NFR-03 | データはMySQLデータベースで管理 |
| NFR-04 | UIはシンプルで直感的に操作できること |
| NFR-05 | 日本語表示対応 |
| NFR-06 | Docker / Docker Composeによる環境構築 |
| NFR-07 | パスワードはbcryptでハッシュ化して保存する |
| NFR-08 | JWTトークンによるセッション管理 |
| NFR-09 | ページ初期表示は3秒以内 |
| NFR-10 | APIレスポンスは1秒以内 |
