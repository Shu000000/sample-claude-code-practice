# 管理者ダッシュボード 基本設計書

**画面ID:** SCR-02
**画面名:** 管理者ダッシュボード
**対象ユーザー:** 管理者
**作成日:** 2026-03-04

---

## 1. 画面概要

管理者がシステム全体の状況を把握し、主要機能へのナビゲーションを行うダッシュボード画面。

---

+----------------------------------------------------------+
|                                                          |

|  +------------------------+   +----------------------+   |

---

## 3. 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| SCR-02-F01 | ナビゲーション | 各機能画面へのリンク |
| SCR-02-F02 | サマリー表示 | 今月のシフト確定・未確定数、従業員数 |
| SCR-02-F03 | クイックアクション | 主要機能への直接アクセス |
| SCR-02-F04 | 直近シフト一覧 | 今週のシフト一覧表示 |
| SCR-02-F05 | ログアウト | セッション終了 |

---

## 4. 画面遷移

| 条件 | 遷移先 |
|------|--------|
| 「シフト管理」クリック | シフト作成画面（SCR-03） |
| 「従業員管理」クリック | 従業員一覧画面（SCR-04） |
| 「希望シフト確認」クリック | 希望シフト確認モーダルまたは画面 |
| 「ログアウト」クリック | ログイン画面（SCR-01） |

---

## 5. 表示データ

| データ | 取得元 |
|--------|--------|
| 今月の確定済みシフト件数 | シフトAPI |
| 今月の未確定シフト件数 | シフトAPI |
| 登録従業員数 | 従業員API |
| 今週のシフト一覧 | シフトAPI |

---

## 6. 関連API

| API | 用途 |
|-----|------|
| GET /api/shifts | シフト一覧取得 |
| GET /api/employees | 従業員一覧取得 |
| POST /api/auth/logout | ログアウト |
