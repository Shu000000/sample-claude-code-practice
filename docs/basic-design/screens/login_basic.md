# ログイン画面 基本設計書

**画面ID:** SCR-01
**画面名:** ログイン画面
**対象ユーザー:** 全ユーザー（管理者・一般従業員）
**作成日:** 2026-03-04

---

## 1. 画面概要

システムへのアクセス認証を行う画面。メールアドレスとパスワードを入力してログインする。

---

## 2. 画面レイアウト

```
+------------------------------------------+
|         シフト管理システム                  |
|                                          |
|  +------------------------------------+  |
|  |  メールアドレス                      |  |
|  |  [___________________________]     |  |
|  |                                    |  |
|  |  パスワード                          |  |
|  |  [___________________________]     |  |
|  |                                    |  |
|  |  [        ログイン         ]        |  |
|  +------------------------------------+  |
|                                          |
+------------------------------------------+
```

---

## 3. 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| SCR-01-F01 | ログインフォーム | メールアドレス・パスワード入力 |
| SCR-01-F02 | ログインボタン | 認証処理実行 |
| SCR-01-F03 | エラーメッセージ表示 | 認証失敗時のメッセージ表示 |

---

## 4. 画面遷移

| 条件 | 遷移先 |
|------|--------|
| 管理者ログイン成功 | 管理者ダッシュボード（SCR-02） |
| 従業員ログイン成功 | シフト確認画面（SCR-06） |
| ログイン失敗 | ログイン画面（エラーメッセージ表示） |

---

## 5. 入力項目

| 項目名 | 型 | 必須 | バリデーション |
|--------|-----|------|--------------|
| メールアドレス | text | 必須 | メール形式 |
| パスワード | password | 必須 | 最低6文字 |

---

## 6. 関連API

| API | 用途 |
|-----|------|
| POST /api/auth/login | ログイン認証 |
