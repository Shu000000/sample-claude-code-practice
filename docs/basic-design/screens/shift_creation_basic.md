# シフト作成画面 基本設計書

**画面ID:** SCR-03
**画面名:** シフト作成画面（カレンダー形式）
**対象ユーザー:** 管理者
**作成日:** 2026-03-04

---

## 1. 画面概要

管理者がカレンダー形式でシフトを作成・編集・管理する画面。月別・週別の切り替えができ、従業員の割り当てと勤務時間を設定できる。

---

```
+----------------------------------------------------------+
| シフト管理システム              管理者: [名前] [ログアウト]  |
+----------------------------------------------------------+
| [シフト管理] [従業員管理] [希望シフト確認]                  |
+----------------------------------------------------------+
|  シフト作成                                               |
|  [< 前月] 2026年3月 [次月>]  [月別|週別]  [シフト確定]     |
|  [新規シフト追加] [CSVエクスポート]                         |
```

---

## 3. 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| SCR-03-F01 | カレンダー表示 | 月別・週別のカレンダー表示 |
| SCR-03-F02 | 月・週切り替え | 月別・週別ビューの切り替え |
| SCR-03-F03 | 前後ナビゲーション | 前月・次月（前週・次週）への移動 |
| SCR-03-F04 | シフト追加 | 日付をクリックしてシフト追加モーダル表示 |
| SCR-03-F05 | シフト編集 | 既存シフトをクリックして編集 |
| SCR-03-F06 | シフト削除 | シフトの削除 |
| SCR-03-F07 | シフト確定 | 月単位でシフトを確定する |
| SCR-03-F08 | CSVエクスポート | 現在表示中のシフトをCSV出力 |

---

## 4. シフト追加・編集モーダル

```
+----------------------------------+
|  シフト追加/編集                   |
|  日付: 2026-03-04               |
|                                  |
|  従業員: [ドロップダウン選択]       |
|  開始時刻: [HH:MM]               |
|  終了時刻: [HH:MM]               |
|                                  |
|  [保存] [キャンセル]               |
+----------------------------------+
```

---

## 5. 画面遷移

| 条件 | 遷移先 |
|------|--------|
| 「管理者ダッシュボード」クリック | 管理者ダッシュボード（SCR-02） |

## 6. 関連API

| API | 用途 |
|-----|------|
| GET /api/shifts | シフト一覧取得 |
| POST /api/shifts | シフト新規作成 |
| PUT /api/shifts/:id | シフト更新 |
| DELETE /api/shifts/:id | シフト削除 |
| PUT /api/shifts/confirm | シフト確定 |
| GET /api/employees | 従業員一覧（ドロップダウン用） |
| GET /api/export/csv | CSVエクスポート |
