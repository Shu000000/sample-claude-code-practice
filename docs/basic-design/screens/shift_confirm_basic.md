# シフト確認画面 基本設計書

**画面ID:** SCR-06
**画面名:** シフト確認画面
**対象ユーザー:** 一般従業員
**作成日:** 2026-03-04

---

## 1. 画面概要

一般従業員が自分の確定済みシフトを確認する画面。月別・週別で表示でき、スマートフォンでの閲覧に対応する。

---

## 2. 画面レイアウト

```
+----------------------------------------------------------+
| シフト管理システム              [名前] さん [ログアウト]    |
+----------------------------------------------------------+
| [シフト確認] [希望シフト入力]                              |
+----------------------------------------------------------+
|  シフト確認                                               |
|  [< 前月] 2026年3月 [次月>]  [月別|週別]                   |
+----------------------------------------------------------+
|  月  |  火  |  水  |  木  |  金  |  土  |  日             |
|------|------|------|------|------|------|------|           |
|  2   |  3   |  4   |  5   |  6   |  7   |  8             |
|      |      |9:00  |      |9:00  |      |                 |
|      |      |~18:00|      |~18:00|      |                 |
+----------------------------------------------------------+
|  今月の勤務日数: X日 / 合計勤務時間: XX時間                 |
+----------------------------------------------------------+
```

---

## 3. 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| SCR-06-F01 | シフト一覧表示 | 自分のシフトをカレンダー形式で表示 |
| SCR-06-F02 | 月・週切り替え | 月別・週別ビューの切り替え |
| SCR-06-F03 | 前後ナビゲーション | 前月・次月（前週・次週）への移動 |
| SCR-06-F04 | 勤務サマリー | 今月の勤務日数・合計勤務時間の表示 |
| SCR-06-F05 | レスポンシブ表示 | スマートフォン対応のレイアウト |

---

## 4. スマートフォン表示

```
+------------------------+
| シフト管理システム      |
| [名前] さん [ログアウト] |
+------------------------+
| [シフト確認][希望入力]  |
+------------------------+
| < 2026年3月 >          |
| 月 火 水 木 金 土 日    |
| 2  3  4  5  6  7  8   |
|       9:00             |
|       ~18:00           |
+------------------------+
| 勤務日数: X日           |
| 合計時間: XX時間         |
+------------------------+
```

---

## 5. 表示データ

| データ | 説明 |
|--------|------|
| 自分の確定シフト | 確定済みのシフトのみ表示（未確定は非表示） |
| 勤務日数 | 当月の勤務予定日数 |
| 合計勤務時間 | 当月の勤務予定時間の合計 |

---

## 6. 関連API

| API | 用途 |
|-----|------|
| GET /api/shifts/my | 自分のシフト一覧取得 |
