# 希望シフト確認画面（管理者用） 基本設計書

**画面ID:** SCR-07
**画面名:** 希望シフト確認画面（管理者用）
**対象ユーザー:** 管理者
**作成日:** 2026-03-04

---

## 1. 画面概要

管理者が全従業員から提出された希望シフトを一覧で確認する画面。月別に絞り込み、シフト作成の参考にできる。

---

## 2. 画面レイアウト

```
+----------------------------------------------------------+
| シフト管理システム              管理者: [名前] [ログアウト]  |
+----------------------------------------------------------+
| [シフト管理] [従業員管理] [希望シフト確認]                  |
+----------------------------------------------------------+
|  希望シフト確認                                           |
|  [< 前月] 2026年4月 [次月>]                               |
|  [従業員フィルター: 全員 ▼]                               |
+----------------------------------------------------------+
|  従業員名   | 日付       | 可否  | 希望時間        | 備考   |
|-------------|-----------|-------|----------------|--------|
|  田中 太郎  | 4/1 (月)  | 希望  | 09:00〜18:00   |        |
|  田中 太郎  | 4/3 (水)  | 不可  | -              | 私用   |
|  鈴木 花子  | 4/1 (月)  | 相談  | -              | 午後可  |
+----------------------------------------------------------+
```

---

## 3. 機能一覧

| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| SCR-07-F01 | 希望シフト一覧表示 | 全従業員の希望シフトを月別表示 |
| SCR-07-F02 | 月切り替え | 前月・次月への移動 |
| SCR-07-F03 | 従業員フィルター | 特定従業員の希望シフトに絞り込み |
| SCR-07-F04 | シフト作成リンク | シフト作成画面へのリンク |

---

## 4. 表示データ

| カラム | 内容 |
|--------|------|
| 従業員名 | 希望シフトを提出した従業員名 |
| 日付 | 希望日（曜日付き） |
| 可否 | 希望あり / 不可 / 相談可 |
| 希望時間 | 希望ありの場合の開始〜終了時刻 |
| 備考 | 従業員が入力した備考 |

---

## 5. 画面遷移

| 条件 | 遷移先 |
|------|--------|
| 「シフト管理」クリック | シフト作成画面（SCR-03） |

---

## 6. 関連API

| API | 用途 |
|-----|------|
| GET /api/shift-requests | 全従業員の希望シフト一覧取得 |
| GET /api/employees | 従業員フィルター用一覧 |
