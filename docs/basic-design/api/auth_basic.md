# 認証API 基本設計書

**API ID:** API-01
**API名:** 認証API
**作成日:** 2026-03-04

---

## 1. 概要

ユーザー認証（ログイン・ログアウト）とJWTトークン管理を行うAPI。

---

## 2. エンドポイント一覧

| メソッド | パス | 説明 | 認証要否 |
|---------|------|------|---------|
| POST | /api/auth/login | ログイン | 不要 |
| POST | /api/auth/logout | ログアウト | 必要 |
| GET | /api/auth/me | ログインユーザー情報取得 | 必要 |

---

## 3. 認証方式

- JWT（JSON Web Token）を使用
- ログイン成功時にアクセストークンを発行
- トークンはAuthorizationヘッダーに Bearer スキームで付与
- トークン有効期限: 24時間

---

## 4. 権限制御

| ロール | 権限 |
|--------|------|
| admin | 全APIへのアクセス可能 |
| employee | 自分のシフト閲覧・希望シフト操作のみ |

---

## 5. エラーコード

| コード | 説明 |
|--------|------|
| 401 | 認証失敗（メール・パスワード不一致） |
| 401 | トークン無効・期限切れ |
| 403 | 権限不足 |
