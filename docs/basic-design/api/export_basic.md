# エクスポートAPI 基本設計書

**API ID:** API-05
**API名:** エクスポートAPI
**作成日:** 2026-03-04

---

## 1. 概要

シフトデータをCSVまたはExcel（xlsx）形式でエクスポートするAPI。管理者のみアクセス可能。

---

## 2. エンドポイント一覧

| メソッド | パス | 説明 | 権限 |
|---------|------|------|------|
| GET | /api/export/csv | シフトをCSV形式でエクスポート | admin |
| GET | /api/export/excel | シフトをExcel（xlsx）形式でエクスポート | admin |

---

## 3. クエリパラメータ

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| year | int | 必須 | 対象年 |
| month | int | 必須 | 対象月 |
| employee_id | int | 任意 | 特定従業員のみ出力 |

---

## 4. レスポンス

- Content-Type: text/csv; charset=utf-8
- Content-Disposition: attachment; filename="shift_{year}{month}.csv"

### CSVフォーマット

```
日付,従業員名,開始時刻,終了時刻,勤務時間,状態
2026-03-04,田中 太郎,09:00,18:00,8.0,確定済み
2026-03-05,鈴木 花子,10:00,17:00,7.0,確定済み
```

---

## 5. エラーコード

| コード | 説明 |
|--------|------|
| 400 | クエリパラメータ不正 |
| 403 | 管理者以外のアクセス |
